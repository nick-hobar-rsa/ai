# AI Development Workspace

This repository serves as a central hub for my AI development research, local model testing, and LLM prompt engineering. 

## Directory Structure

* **`chatgpt/`**, **`claude/`**, **`gemini/`**: Documentation, prompt chains, and architecture handoffs for various cloud-based LLMs. Includes structured frameworks like the QRSPI planning method.
* **`karpathy/`**: Implementations and experiments inspired by Andrej Karpathy's research (e.g., auto-research scripts).
* **`llama.cpp/`**: Port of the LLaMA model in C/C++ for local inference. 
* **`ollama-gemma4-31b/`**: Research and codebase analysis utilizing the Gemma architecture via Ollama.
* **`opencode/`**, **`openclaw/`**: Open-source AI tooling and Qwen-based codebase research.
* **`zai-glm-5/`**: Project planning and codebase research centered around the GLM-5 model.
